### 프로젝트 소개

'시간은 금이다'는 사용자가 작심삼일을 벗어날 수 있도록 시간을 효율적으로 관리할 수 있는 도구입니다.<br/>
쉽고 직관적인 방법으로 사용자들의 일정을 관리할 수 있도록 월간 & 주간 & 일간 캘린더, Todo 설정, 중요 Todo 설정 기능을 제공합니다.
